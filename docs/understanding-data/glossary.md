# Glossary of Key Terms

Understanding the terminology used in African language data collection and NLP research.

## General Terms

**African Language NLP**: Natural Language Processing research and applications focused on African languages.

**Community-based Data Collection**: Collaborative approaches that involve native speakers as partners in creating language resources.

**Low-resource Language**: A language with limited digital resources, computational tools, or research attention.

**Orthography**: The conventional spelling system of a language.

## Voice Data Terms

**Automatic Speech Recognition (ASR)**: Technology that converts spoken language into text.

**Forced Alignment**: The process of automatically aligning audio with its corresponding transcript at the word or phoneme level.

**Sampling Rate**: The frequency at which audio is digitally recorded, typically measured in Hz or kHz.

**Text-to-Speech (TTS)**: Technology that converts written text into spoken language.

## Text Data Terms

**Corpus** (plural: Corpora): A large collection of texts used for linguistic analysis or machine learning.

**Named Entity Recognition (NER)**: The task of identifying and classifying named entities (people, places, organizations) in text.

**Part-of-Speech (POS) Tagging**: The process of marking words with their grammatical categories.

**Tokenization**: The process of breaking text into individual words or meaningful units.

## Translation Terms

**Parallel Corpus**: A collection of texts in two or more languages that are translations of each other.

**Machine Translation (MT)**: Automated translation of text from one language to another.

**BLEU Score**: A metric for evaluating the quality of machine-translated text against reference translations.

**Back-translation**: The process of translating text back to its original language to check translation quality.

## Data Quality Terms

**Inter-annotator Agreement**: A measure of how consistently multiple annotators assign the same labels to data.

**Data Validation**: The process of checking data for accuracy, completeness, and consistency.

**Quality Assurance (QA)**: Systematic processes to ensure data meets specified standards.

## Technical Terms

**API**: Application Programming Interface - a way for different software applications to communicate.

**JSON**: JavaScript Object Notation - a lightweight data interchange format.

**Metadata**: Data that provides information about other data.

**Version Control**: Systems for tracking changes to files over time (e.g., Git).
